# Cold Tech Dossier Visual Design

## Direction

The DC-Agent interface will move from a black-gold dossier style to a colder, more technical intelligence-system style. The target feeling is a premium database search terminal: titanium gray surfaces, frosted translucent panels, ice-blue system highlights, and restrained amber only for warnings or classified accents.

## Palette

- Base: deep titanium, graphite, blue-black.
- Surface: translucent slate and cool glass.
- Primary accent: ice blue / cyan.
- Secondary accent: muted violet-blue for neural background depth.
- Warning accent: very limited amber, used only where classification or caution needs emphasis.

## UI Treatment

- Main panels should feel lighter than the current black-gold UI, with more visible background motion.
- Borders shift from gold to cool cyan/slate.
- Buttons and active states use cyan highlights instead of gold gradients.
- Text hierarchy stays calm: off-white primary text, blue-gray secondary text.
- Three.js quantum neural network remains the background anchor, with less heavy dark overlay.

## Scope

This pass changes visual styling only. It does not rename API contracts, change data flow, alter chat/search behavior, or add new panels.
